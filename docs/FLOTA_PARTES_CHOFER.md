# Partes diarios (Chofer)

Este flujo replica el formulario que hoy usan (coche + km + textos por área + fotos + combustible en ruta).

URL:
- `/flota/chofer/partes/nuevo/`

Implementación:
- Se guardan textos en campos separados para reportes.
- `descripcion` se autogenera para compatibilidad con listados existentes.
- Fotos se guardan como adjuntos del parte.
