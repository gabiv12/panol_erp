# Informe de flota

## Objetivo
Una pantalla única para ver el estado general:
- Salidas próximas (ventana configurable)
- Partes abiertos/en proceso por unidad (alerta)
- Vencimientos (VTV y matafuegos)
- Carga por chofer (cantidad de salidas en el día)

## Uso típico
- Al iniciar el turno (diagramador/administración): revisar alertas y prioridades.
- Antes de armar el diagrama: ver si alguna unidad tiene partes abiertos.

## Notas
- El informe NO bloquea: solo informa.
- Offline-first: sin dependencias externas.
